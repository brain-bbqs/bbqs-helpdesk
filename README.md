# BBQS Helpdesk

A helpdesk for any and all questions, BBQS-specific, or neuroscience in general.

[Discussions](https://github.com/brain-bbqs/bbqs-helpdesk/discussions) may be transferred to other helpdesks as needed in the search for answers.

For example, [EMBER-Pennsieve](https://github.com/brain-bbqs/ember-pennsieve-helpdesk) questions may also be raised directly on that helpdesk if your question involves how to use the platform or if you are requesting a new data processing workflow.
